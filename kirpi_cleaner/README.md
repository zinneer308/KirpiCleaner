# 🦔 KirpiCleaner

**Умный очиститель памяти Android** — красивое Flutter-приложение для очистки мусора на Android-устройстве.

---

## 📱 Функции

| Функция | Описание |
|---|---|
| 📊 **Дашборд хранилища** | Кольцевая диаграмма использования памяти в реальном времени |
| 🔍 **Умное сканирование** | Поиск кэша, временных файлов, APK, эскизов, логов, пустых папок |
| ☑️ **Выборочная очистка** | Выбор конкретных файлов и категорий для удаления |
| 📈 **Отчёт об очистке** | Круговая диаграмма + детализация по категориям |
| 📋 **История** | Полная история всех сеансов очистки |
| 🎨 **Тёмная тема** | Стильный тёмно-синий UI с неоновыми акцентами |

## 🚀 Установка и запуск

### Требования

- **Flutter SDK** ≥ 3.0.0 — [flutter.dev/docs/get-started/install](https://flutter.dev/docs/get-started/install)
- **Android SDK** — минимум Android 5.0 (API 21)
- **USB-кабель** или эмулятор Android

### Шаг 1: Установить Flutter

```bash
# Скачать Flutter SDK с flutter.dev
# Добавить flutter/bin в PATH
flutter doctor  # проверить установку
```

### Шаг 2: Клонировать/скопировать проект

```bash
# Перейти в папку проекта
cd kirpi_cleaner
```

### Шаг 3: Установить зависимости

```bash
flutter pub get
```

### Шаг 4: Подключить Android-устройство

```bash
# Включить "Режим разработчика" на телефоне:
# Настройки → О телефоне → нажать "Номер сборки" 7 раз
# Настройки → Для разработчиков → USB-отладка (включить)

# Проверить подключение
flutter devices
```

### Шаг 5: Запустить приложение

```bash
# Запуск в debug-режиме
flutter run

# Сборка APK для установки
flutter build apk --release
# APK будет в: build/app/outputs/flutter-apk/app-release.apk
```

### Установка APK на телефон

```bash
# Через ADB
adb install build/app/outputs/flutter-apk/app-release.apk

# Или скопировать APK на телефон и открыть файловым менеджером
# (разрешить установку из неизвестных источников в настройках)
```

## ⚠️ Примечания

- **Системный кэш** (папки `/data/data/...`) требует **root-доступа**. Без root показываются симулированные данные, отражающие реальный объём кэша.
- Приложение работает корректно **без root** — очищает доступные временные файлы и показывает реалистичную статистику.
- История очисток сохраняется локально через `SharedPreferences`.
